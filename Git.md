

### Comandos:

```
git init -> Inicializar o repositório
git config -> Configurações de usuário
git config --local --global  -> diz respeito a quem pode modificar os arquivos

git remote -> diz respeito ao repositório na nuvem 
git add -> Adiciona as modificações para uma "fila"
git commit -> Confirma as modificações na fila
git branch -> Controla as ramificações do repositório
git push -> Salva as modificações que foram confirmadas
git pull -> Puxa o repositório na nuvem para comparar com o local

git clone -> Clona um repositório da nuvem (pode ser um repositório nosso ou de outro usuário)
```


### Fluxo do git 
```
1 - Inicializo o repositório
2 - Se for a primeira vez, configuro usuário
3 - Se for a primeira vez, configuro o repositório na nuvem
4 - Adiciono as modificações na fila
5 - Confirmo as modificações
6 - Empurro as modificações para nuvem
```
